## Pengenalan

PT DUTA COMPUTER Sistem Manajemen Absensi adalah aplikasi web dan mobile untuk mengelola kehadiran karyawan dengan fitur-fitur modern seperti:
- Check-in/Check-out dengan GPS
- Riwayat absensi
- Laporan kehadiran
- Manajemen cuti
- Profil karyawan

## Login

### Cara Login

1. Buka aplikasi di browser atau aplikasi mobile
2. Masukkan email Anda
3. Masukkan password Anda
4. Klik tombol "Masuk"

### Lupa Password

1. Klik link "Lupa password?" di halaman login
2. Masukkan email Anda
3. Cek email untuk link reset password
4. Ikuti instruksi di email

### Keamanan Login

- Jangan bagikan password Anda kepada siapa pun
- Gunakan password yang kuat (minimal 8 karakter)
- Logout setelah selesai menggunakan aplikasi
- Aktifkan 2FA (Two-Factor Authentication) untuk keamanan ekstra

## Dashboard

### Tampilan Dashboard

Dashboard menampilkan:
- **Salam Pagi**: Ucapan sesuai waktu
- **Status Hari Ini**: Check-in dan check-out status
- **Aksi Cepat**: Tombol untuk absensi, riwayat, laporan, profil
- **Saldo Cuti**: Sisa cuti tahunan, sakit, dan khusus
- **Riwayat Terbaru**: 7 hari terakhir absensi

### Navigasi

Gunakan menu di bawah (mobile) atau sidebar (web) untuk navigasi:
- Dashboard
- Absensi
- Riwayat
- Laporan
- Profil

## Absensi

### Check-in

1. Buka halaman Absensi
2. Klik tombol "Check-in Sekarang"
3. Sistem akan mengambil lokasi Anda
4. Konfirmasi lokasi
5. Check-in berhasil

**Catatan**: 
- Check-in hanya bisa dilakukan sekali per hari
- Lokasi akan dicatat untuk verifikasi
- Jika check-in setelah jam 08:00, status akan "Terlambat"

### Check-out

1. Buka halaman Absensi
2. Klik tombol "Check-out Sekarang"
3. Sistem akan mengambil lokasi Anda
4. Konfirmasi lokasi
5. Check-out berhasil

**Catatan**:
- Check-out hanya bisa dilakukan setelah check-in
- Jam kerja akan dihitung otomatis
- Overtime akan dicatat jika bekerja lebih dari 8 jam

### Troubleshooting

**Lokasi tidak terdeteksi**
- Pastikan GPS aktif di device
- Berikan izin akses lokasi ke aplikasi
- Coba perbarui lokasi secara manual

**Tidak bisa check-in**
- Pastikan sudah login
- Pastikan belum check-in hari ini
- Cek koneksi internet

## Riwayat Absensi

### Melihat Riwayat

1. Buka halaman Riwayat
2. Scroll untuk melihat daftar absensi
3. Klik item untuk melihat detail

### Filter Riwayat

1. Gunakan filter tanggal (jika tersedia)
2. Pilih bulan dan tahun
3. Klik "Terapkan Filter"

### Informasi yang Ditampilkan

- **Tanggal**: Hari dan tanggal absensi
- **Check-in**: Waktu check-in
- **Check-out**: Waktu check-out
- **Status**: Present, Late, Absent, Work Leave
- **Jam Kerja**: Total jam kerja
- **Overtime**: Jam kerja tambahan

## Laporan

### Melihat Laporan

1. Buka halaman Laporan
2. Pilih bulan dan tahun
3. Lihat statistik kehadiran

### Statistik yang Ditampilkan

- **Total Hari**: Jumlah hari kerja
- **Hadir**: Jumlah hari hadir
- **Terlambat**: Jumlah hari terlambat
- **Tidak Hadir**: Jumlah hari tidak hadir
- **Cuti Kerja**: Jumlah hari cuti kerja
- **Total Jam Kerja**: Total jam kerja dalam bulan
- **Total Overtime**: Total jam overtime

### Export Laporan

1. Buka halaman Laporan
2. Klik tombol "Export"
3. Pilih format (PDF atau CSV)
4. File akan diunduh otomatis

## Profil

### Melihat Profil

1. Buka halaman Profil
2. Lihat informasi pribadi Anda

### Informasi Profil

- **Nama**: Nama lengkap
- **Email**: Email perusahaan
- **ID Karyawan**: Nomor identitas karyawan
- **Nomor Telepon**: Nomor telepon
- **Tanggal Lahir**: Tanggal lahir
- **Jenis Kelamin**: Laki-laki/Perempuan
- **Alamat**: Alamat rumah
- **Shift**: Shift kerja Anda

### Edit Profil

1. Klik tombol "Edit Profil"
2. Ubah informasi yang ingin diubah
3. Klik "Simpan"

### Ubah Password

1. Klik tombol "Ubah Password"
2. Masukkan password lama
3. Masukkan password baru (minimal 8 karakter)
4. Konfirmasi password baru
5. Klik "Ubah"

### Saldo Cuti

Lihat saldo cuti Anda:
- **Cuti Tahunan**: Cuti yang diberikan setiap tahun
- **Cuti Sakit**: Cuti untuk keperluan kesehatan
- **Cuti Khusus**: Cuti untuk keperluan khusus

## FAQ

### Pertanyaan Umum

**Q: Berapa jam kerja normal?**
A: Jam kerja normal adalah 8 jam per hari (08:00 - 17:00).

**Q: Apa itu overtime?**
A: Overtime adalah jam kerja tambahan di luar jam kerja normal. Jika Anda bekerja lebih dari 8 jam, jam tambahan akan dicatat sebagai overtime.

**Q: Bagaimana cara mengajukan cuti?**
A: Hubungi bagian HR atau gunakan fitur pengajuan cuti di aplikasi (jika tersedia).

**Q: Apa yang harus saya lakukan jika lupa check-in?**
A: Hubungi bagian HR untuk melakukan adjustment manual.

**Q: Apakah saya bisa check-in dari rumah?**
A: Tergantung kebijakan perusahaan. Hubungi HR untuk informasi lebih lanjut.

**Q: Bagaimana jika lokasi GPS tidak akurat?**
A: Sistem akan mencatat lokasi terbaik yang tersedia. Jika ada masalah, hubungi IT Support.

**Q: Berapa lama data absensi disimpan?**
A: Data absensi disimpan selama 7 tahun sesuai kebijakan perusahaan.

**Q: Bagaimana cara logout?**
A: Klik menu profil dan pilih "Logout".

### Troubleshooting

**Q: Aplikasi tidak bisa login**
A: 
- Pastikan email dan password benar
- Cek koneksi internet
- Coba refresh halaman
- Hubungi IT Support jika masalah berlanjut

**Q: Lokasi tidak terdeteksi**
A:
- Pastikan GPS aktif
- Berikan izin akses lokasi
- Coba perbarui lokasi
- Gunakan WiFi jika GPS tidak bekerja

**Q: Laporan tidak muncul**
A:
- Refresh halaman
- Cek filter tanggal
- Coba bulan lain
- Hubungi IT Support
